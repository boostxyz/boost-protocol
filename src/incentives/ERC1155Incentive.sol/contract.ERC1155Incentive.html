Boost Protocol Docs

ERC1155Incentive

Git Source

Inherits: Incentive, IERC1155Receiver

A simple ERC1155 incentive implementation that allows claiming of tokens

State Variables

asset

The address of the ERC1155-compliant contract

IERC1155 public asset;

strategy

The strategy for the incentive (MINT or POOL)

Strategy public strategy;

limit

The maximum number of claims that can be made (one per address)

uint256 public limit;

tokenId

The ERC1155 token ID for the incentive

uint256 public tokenId;

extraData

Extra data to be passed to the ERC1155 contract

bytes public extraData;

Enums

Strategy

The strategy for the incentive

*The strategy determines how the incentive is disbursed:

  • POOL: Transfer tokens from the pool to the recipient
  • MINT: Mint tokens to the recipient directly (not yet implemented)*
enum Strategy {
    POOL,
    MINT
}
