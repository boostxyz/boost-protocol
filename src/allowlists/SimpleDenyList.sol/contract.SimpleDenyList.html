SimpleDenyList

Git Source

Inherits: AllowList

A simple implementation of an AllowList that implicitly allows all addresses except those explicitly added to the deny list

State Variables

_denied

An internal mapping of denied statuses

mapping(address => bool) private _denied;

Functions

constructor

Construct a new SimpleDenyList

Because this contract is a base implementation, it should not be initialized through the constructor. Instead, it should be cloned and initialized using the initialize function.

constructor();
